## 5.2 Invoice Requirements (Obligations Légales de Facturation)

## 5.3 VAT Specifics (TVA Tunisie)

Standard rates:
- 19%: Default rate for most goods and services
- 13%: Reduced rate (construction, tourism, certain services)
- 7%: Super-reduced rate (medical, educational, essential goods)

Special regimes:
- Export: 0% VAT (exemption with right to deduct)
- Suspension: VAT suspension for certain industries
- Franchise en base: Small business exemption threshold

Factarlou handles all regimes with correct millimes rounding.

## 5.4 Withholding Tax Matrix (Retenue à la Source)

| Rate | Applicability |
|------|---------------|
| 0.5% | Certain merchandise sales |
| 1%   | Standard B2B services |
| 1.5% | Specific professional services |
| 5%   | Rental income, certain royalties |
| 10%  | Non-resident services, some commissions |
| 15%  | Dividends, certain interest payments |

Factarlou auto-selects based on transaction type and allows manual override.

## 5.5 TEJ Portal Workflow

Step 1: User completes invoices/RS certificates in Factarlou
Step 2: User selects period (month/quarter)

Step 4: Factarlou generates XML file per DGI schema
Step 5: User uploads XML to https://tej.finances.gov.tn
Step 6: TEJ validates and returns acceptance/rejection
Step 7: Factarlou logs submission status per document
