📚 Factarlou — Documentation Complète

Bienvenue dans la documentation officielle de Factarlou, la première application de bureau intelligente pour la facturation et la gestion d'entreprise conçue pour le marché tunisien.

1. Aperçu

Factarlou est une application de bureau haute performance, "offline-first", construite avec Electron.js. Elle est conçue pour gérer l'intégralité du cycle financier des freelances, PME et startups tunisiennes — de la génération de devis au suivi des dépenses et à la conformité fiscale — tout en garantissant une confidentialité des données à 100%.

2. Fonctionnalités Clés

🏢 Facturation de Base (Invoicing)

  • Types de Documents : Créez des Factures professionnelles, des Devis et des Bons de Commande.
  • Devis en Facture : Conversion en un clic d'un Devis en Facture avec mise à jour automatique du statut.
  • Logique Fiscale Tunisienne :
    • Calcul automatique du Timbre Fiscal (1.000 TND).
    • Support pour plusieurs taux de TVA (0%, 7%, 13%, 19%).
    • Gestion du FODEC et autres taxes spécifiques.
  • Formatage des Nombres : Support des normes comptables tunisiennes (jusqu'à 3 décimales) avec ajustements automatiques des arrondis.

🧠 Scanner de Dépenses Intelligent (OCR)

  • Saisie Automatisée : Utilise Tesseract.js pour scanner les reçus papier et les PDF.
  • Extraction de Données : Détecte automatiquement le nom du fournisseur, la date et le montant total (TTC).
  • Texte en Nombre : Conversion intelligente des montants écrits en mots (ex: "deux cent dinars" → 200.000).
  • Gestion des Pièces Jointes : Chaque dépense est liée à son scan numérique pour un audit facile.

🇹🇳 Conformité Fiscale & Exports XML

  • Certificats Automatiques : Générez des certificats officiels de "Retenue à la Source" basés sur la loi tunisienne actuelle.
  • Préréglages de Taux : Support intégré pour les taux de 0,5%, 1%, 1,5%, 5%, 10% et 15%.
  • Export XML (Portail TEJ) : Générez des fichiers XML structurés pour les déclarations de retenues à la source (RS) et les factures (TEIF), prêts à être téléchargés sur le portail de l'administration fiscale.
  • Exportation par Lot : Visualisez et exportez les résumés de toutes les retenues fiscales pour une période spécifique.

🤝 Gestion Clients & Contrats

  • CRM Lite : Gère une base de données complète de clients avec historique et coordonnées.
  • Aperçu Intelligent : Vue détaillée instantanée des profils clients.
  • Générateur de Contrats : Générez divers contrats de travail et de service tunisiens (CDI, CDD, Freelance, etc.) à l'aide de modèles professionnels.

🎨 Personnalisation & Branding

  • Identité Visuelle : Importez votre logo, votre cachet numérique et votre signature.
  • Visuels Automatiques : Vos visuels sont automatiquement intégrés à vos factures, devis et certificats de retenue.
  • Barre d'Accent : Personnalisez les éléments graphiques pour qu'ils correspondent à votre charte graphique.

📊 Analyses en Temps Réel

  • Tableau de Bord : Graphiques interactifs montrant les revenus vs les dépenses.
  • Suivi de Statut : Indicateurs visuels pour les documents non payés, partiellement payés et payés.
  • Métriques Clés : Identifiez vos clients les plus précieux et vos catégories de dépenses les plus élevées.

3. Architecture Technique

🛠️ Stack Technique

  • Runtime : Electron.js (Support multiplateforme Windows, macOS, Linux).
  • Frontend : Vanilla JS, HTML5, CSS3 (Pas de frameworks lourds pour une performance maximale).
  • Base de données : SQLite (via better-sqlite3) pour un stockage de données local et robuste.
  • Moteur OCR : Tesseract.js (Reconnaissance Optique de Caractères).
  • Moteur PDF : Processus d'arrière-plan Electron isolé pour un rendu A4 parfait au pixel près.

🔒 Confidentialité & Sécurité

  • Local-First : Toutes les données financières, listes de clients et mots de passe restent sur votre machine physique.
  • Chiffrement : Les mots de passe sont hachés avec bcrypt avant stockage.
  • Pas de Télémétrie : Pas de suivi, pas d'analyses, pas de téléchargements cloud.

4. Fonctionnement Interne

Le Moteur de Génération PDF

Contrairement aux applications web traditionnelles, Factarlou utilise un processus d'arrière-plan spécialisé pour générer les PDF. Cela garantit que l'interface utilisateur (barres latérales, boutons) n'apparaît jamais sur le document final, ce qui donne un fichier A4 parfaitement professionnel.

Persistance des Données

Les données sont stockées dans le dossier d'application système de l'utilisateur (userData).

  • macOS : ~/Library/Application Support/factarlou/
  • Windows : %AppData%\factarlou\

Cela garantit que même lorsque l'application est mise à jour, vos données restent intactes et en sécurité.

Système de Mise à Update Automatique

L'application est intégrée aux GitHub Releases. Chaque fois qu'une nouvelle version est publiée, l'application la détecte, la télécharge en arrière-plan et invite l'utilisateur à redémarrer pour appliquer les dernières mises à jour fiscales ou nouvelles fonctionnalités.

5. Prise en Main

Installation

  1. Téléchargez le dernier installateur pour votre OS (.dmg pour Mac, .exe for Windows).
  2. Installez comme n'importe quelle autre application de bureau.
  3. Inscription : Créez votre premier utilisateur local (ces données ne quittent jamais votre ordinateur).
  4. Configuration : Allez dans "Mon Entreprise" pour télécharger votre logo, votre cachet et votre signature.

Flux de Travail Quotidien

  1. Créer Client → Générer Devis → Convertir en Facture lors de l'acceptation.
  2. Scanner Reçu → Confirmer les Données → Enregistrer comme Achat.
  3. Vérifier le Tableau de Bord périodiquement pour surveiller les factures impayées et la croissance mensuelle.
