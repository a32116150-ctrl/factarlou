📚 Factarlou — Documentation Complète

Bienvenue dans la documentation officielle de Factarlou v3.1.0, l'ERP Local-First intelligent pour la facturation, le Point de Vente et la gestion d'entreprise — conçu exclusivement pour le marché tunisien.

1. Aperçu

Factarlou est une application de bureau haute performance, "offline-first", construite avec Electron.js. Elle gère l'intégralité du cycle financier des commerces de détail, freelances, PME et startups tunisiennes — de la caisse enregistreuse (POS) à la génération de factures, en passant par la paie, la conformité fiscale et l'export XML TEJ.

2. Fonctionnalités Clés

🛒 Point de Vente (POS) — Caisse Enregistreuse

  • Grille Produits Tactile : Interface optimisée pour le toucher (targets 44px+). Navigation par catégories, favoris et top-10.
  • Scan Code-Barres : Ajout rapide de produits par scan.
  • Paiements Multiples & Split Tender : Espèces, carte, mobile money, chèque. Fractionnez un paiement sur plusieurs moyens.
  • Gestion de Sessions : Ouverture/fermeture de caisse, rapports X/Z, mouvements de fonds.
  • Multi-Caissier : Utilisateurs séparés avec suivi individuel.
  • Avoirs & Remboursements : Retours et remboursements depuis l'interface caisse.
  • Paniers Suspendus : Mettez en attente, reprenez, gérez plusieurs brouillons.
  • Impression Thermique : Tickets sur imprimante 80mm.
  • Raccourcis Clavier : F1-F8 pour toutes les actions POS.

📄 Moteur de Documents (8 Types)

  • Types supportés : Facture, Avoir, Devis, Bon de Commande, Bon de Livraison, Bon d'Entrée, Bon de Sortie, Bon d'Échange.
  • Pipeline Commercial : Suivi Devis → Facture → BL avec référencement automatique des documents liés.
  • Conversion BA → Dépense : Transformez un Bon d'Achat en dépense en un clic.
  • Duplication & Conversion : Dupliquez n'importe quel document ou convertissez-le vers un autre type.
  • Templates : Sauvegardez/chargez des configurations de formulaire réutilisables.
  • Glisser-Déposer : Réorganisez les lignes d'articles par glisser-déposer.
  • Notes Internes : Ajoutez des notes invisibles sur le PDF mais visibles dans l'app.
  • Champs Personnalisés : Paires clé/valeur pour informations spécifiques.

🧠 Scanner OCR Intelligent

  • Saisie Automatisée : Utilise Tesseract.js pour scanner les reçus papier et PDF.
  • Extraction de Données : Détecte automatiquement le nom du fournisseur, la date et le montant total (TTC).
  • Texte en Nombre : Conversion intelligente des montants écrits en mots ("deux cent dinars" → 200.000).
  • Bilingue FR + AR : Support du français et de l'arabe dans la reconnaissance.
  • Cache Persistant : Les analyses suivantes sont quasi-instantanées.

🇹🇳 Conformité Fiscale & Exports XML (Portail TEJ)

  • Multi-Taux TVA : 0%, 7%, 13%, 19% sur un même document (précision 3 décimales).
  • Timbre Fiscal : Calcul automatique à 1.000 TND.
  • Retenue à la Source : Certificats de 0,5% à 15%, conformes à la législation.
  • FODEC : Gestion de la taxe FODEC incluse.
  • Export XML TEJ :
    • Retenues à la Source (RS) : Fichiers prêts pour déclaration mensuelle.
    • Facturation Électronique (TEIF) : Structure XML compatible avec l'administration fiscale.
  • Validateur MF : Validation du Matricule Fiscal en direct.
  • RNE Search : Auto-complétion depuis le Registre National des Entreprises.

📊 Assistant TVA Déclaratif

  • Déclaration Mensuelle : TVA collectée et déductible ventilée par taux.
  • Rapport TVA Annuel : Mois par mois avec cumuls annuels.
  • Simulateur de Scénarios : Comparatif 3 colonnes pour projections fiscales.
  • Calculateur IRPP : Simulation d'impôt sur le revenu.
  • Calculateur de Pénalités : Estimation des pénalités de retard.

👔 Paie & Ressources Humaines

  • Base Employés : Gestion des employés avec contrats (CDI, CDD, CIVP).
  • Fiches de Paie : Génération automatique avec cotisations CNSS (@ 9.18%).
  • Générateurs de Contrats : Modèles professionnels tunisiens (CDI, CDD, Freelance).

🤝 Gestion Clients (CRM)

  • Base Complète : Historique, coordonnées, paiements.
  • Carte Interactive : Visualisation clients sur Leaflet/OpenStreetMap.
  • Générateur de Contrats : Contrats de travail et de service tunisiens.
  • Recherche Globale : Recherche clients et services depuis toutes les pages.

🎨 Personnalisation & Branding

  • Identité Visuelle : Logo, cachet numérique, signature.
  • Design Premium : Thème sombre avec ombres, dégradés, effets hover et système de design complet.
  • Icônes Lucide SVG : Plus de 50 icônes vectorielles remplaçant les anciens émojis.
  • Paramètres Repensés : Sidebar verticale à 5 onglets, mise en page par cartes.

📊 Analyses & Tableau de Bord

  • Graphiques Revenus/Dépenses : Canvas natif, aucune dépendance.
  • Top 5 Clients : Barres horizontales des meilleurs clients.
  • Suivi des Impayés : Liste des factures en retard avec montants.
  • Auto-Redimensionnement : Via ResizeObserver.

📌 Notes & Productivité

  • Notes Autocollantes : Code couleur, épinglage, widget tableau de bord.
  • Templates Email : Sauvegarde/chargement de sujets + corps d'email.
  • Envoi Groupé : Email par lot avec barre de progression.
  • Saisie Semi-Automatique : Auto-complétion et suggestions.

📁 Import / Export

  • CSV : Import/export clients et services.
  • XLSX : Export documents, clients, services.
  • Sauvegarde SQLite : Export complet de la base de données en un clic.
  • Opérations par Lot : Suppression multi-sélection, marquage payé groupé, export PDF batch.

🔄 Opérations Avancées

  • Tri par Colonnes : Cliquez sur un en-tête pour trier.
  • Pagination : 50 documents par page.
  • Factures Récurrentes : Génération automatique (cron 60 min).
  • Sauvegarde Auto des Brouillons : Debounce 2s vers localStorage.
  • Journal d'Audit : Historique des modifications.

3. Architecture Technique

🛠️ Stack Technique

  • Runtime : Electron.js (Windows, macOS, Linux).
  • Frontend : Vanilla JS, HTML5, CSS3 (Pas de frameworks lourds).
  • Base de données : SQLite via better-sqlite3.
  • Moteur OCR : Tesseract.js.
  • Moteur PDF : Processus Electron isolé pour rendu A4 pixel-perfect.
  • Moteur Mathématique : math-utils.js — calculs TVA/taxes centralisés.
  • Validation : validate.js — couche de validation IPC avec messages d'erreur en français.

🔒 Confidentialité & Sécurité

  • Local-First : Toutes les données restent sur votre machine.
  • Chiffrement : Mots de passe hachés avec bcrypt.
  • Pas de Télémétrie : Aucun suivi, analytics ou upload cloud.
  • Migrations Sécurisées : Base de données migrée sans DROP TABLE (modèle RENAME).
  • ON DELETE CASCADE : en JavaScript pour clients et documents.

💼 Positionnement ERP Tunisie

Contrairement aux ERP traditionnels (Odoo, Sage) qui nécessitent des serveurs distants et des abonnements, Factarlou est un ERP Local-First : puissant, léger, rapide et totalement souverain.

4. Fonctionnement Interne

Moteur de Génération PDF

Un processus d'arrière-plan Electron isolé génère les PDF. L'interface utilisateur (barres latérales, boutons) n'apparaît jamais sur le document final.

Persistance des Données

Données stockées dans le dossier userData de l'OS :

  • macOS : ~/Library/Application Support/factarlou/
  • Windows : %AppData%\factarlou\

Les données restent intactes après mise à jour. Sauvegarde automatique incluant les pièces jointes.

Système de Mise à Jour Automatique

  • Détection : Vérification GitHub Releases au démarrage.
  • Téléchargement : Barre de progression (Dock macOS / Taskbar Windows).
  • Installation : Windows : auto-install avec redémarrage. macOS : copie DMG vers Downloads + instructions.
  • Fiabilité v3.1.0 : Correction des faux positifs, messages d'erreur visibles, garde isDestroyed().

Raccourcis Clavier

  • Cmd/Ctrl + N : Nouveau document.
  • Cmd/Ctrl + Shift + N : Nouveau client.
  • Cmd/Ctrl + F : Recherche globale.
  • F1-F8 : Actions POS.

5. Prise en Main

Installation

  1. Téléchargez le dernier installateur (.dmg Mac / .exe Windows) depuis factarlou.online.
  2. Installez comme une application de bureau standard.
  3. Inscription : Créez votre compte local (données 100% sur votre machine).
  4. Configuration : Allez dans "Mon Entreprise" → logo, cachet, signature.

Flux de Travail Quotidien

  1. Commerce (POS) : Ouvrez une session caisse → Scannez produits → Encaissez → Ticket thermique.
  2. Bureau (Documents) : Créer Client → Devis → Convertir en Facture → BL → Suivi paiement.
  3. Dépenses : Scanner Reçu → OCR extrait les données → Enregistrer comme Achat.
  4. Fiscal : Fin de mois → Assistant TVA → Export XML TEJ → Déclaration.
  5. Suivi : Tableau de bord → Vérifier impayés → Analyser tendances.
