🛡️ Politique de Confidentialité

Principe Fondamental : Vos données vous appartiennent. Factarlou est une application "offline-first". Cela signifie qu'aucune de vos données financières, clients ou documents ne quitte jamais votre ordinateur.

1. Collecte des Données

Factarlou ne collecte aucune donnée personnelle ou commerciale. Toutes les informations que vous saisissez dans l'application (noms de clients, montants de factures, scans de reçus) sont stockées localement sur votre disque dur dans une base de données SQLite sécurisée.

2. Stockage Local

L'application utilise le dossier système suivant pour stocker vos informations :

  • macOS : ~/Library/Application Support/factarlou/
  • Windows : %AppData%\factarlou\

Nous n'avons aucun accès à ces fichiers. Seul vous (ou toute personne ayant accès à votre session d'ordinateur) pouvez consulter ces données.

3. Télémétrie et Tracking

Contrairement à la plupart des logiciels modernes, Factarlou n'inclut aucun outil de télémétrie (Google Analytics, Mixpanel, etc.). Nous ne savons pas combien de factures vous créez, ni à quelle fréquence vous utilisez l'application.

4. Services Tiers

L'application n'utilise qu'une seule connexion internet : le module de mise à jour automatique via notre infrastructure sécurisée. Cette connexion ne sert qu'à vérifier si une nouvelle version est disponible et à la télécharger. Aucune information privée n'est transmise lors de cette vérification.

5. Sécurité

Vos mots de passe pour les comptes multi-utilisateurs sont hachés à l'aide de l'algorithme bcrypt. Cela signifie que même si quelqu'un accède physiquement à votre base de données SQLite, il ne pourra pas lire vos mots de passe en clair.
